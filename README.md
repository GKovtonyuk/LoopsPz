# LoopsPz
# 🚀 Mission: C++ Loops Laboratory

## 📖 Історія

У цифровій лабораторії **Algorithm Lab** стався критичний збій системи обчислень.  
Основні програмні модулі перестали працювати, а сервер **Core Processor** перейшов у аварійний режим.

Ви — **інженер-програміст**, якому доручено відновити роботу лабораторії.  
Щоб перезапустити систему, потрібно виконати **5 місій**, використовуючи **цикли C++**.

Кожна місія активує один із модулів системи. Після активації всіх модулів лабораторія буде відновлена.

---

## 🎯 Мета роботи

- Навчитися використовувати **цикли в C++**  
- Обчислювати суми, добутки та статистичні показники  
- Табулювати функції  
- Аналізувати числові послідовності
